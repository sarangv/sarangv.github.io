Minesweeper AI

April 2020 - June 2020

Utilizing my knowledge gained from the Intro. to AI course I took, I developed a program to solve various Minesweeper boards. Minesweeper's objective is to clear a rectangular board containing hidden "mines" without uncovering any of them, with help from clues about the number of neighboring mines in each field. The AI topics used in my projet include Heuristic Problem Solving, Probability, and Predicate Logic. The first approach my program would take it to use the Minimum Remaining Value Heuristic. What this meant was to find spots with a guarantee that they were not mines and uncover them first. Next, for those that could not guarantee it would use Predicate Logic to reason out which spots were mine and which were not. Finally, if neither of those methods resolved the issue, a “smart guessing” algorithm based on probability was used. This last implementation was the most crucial as a large number of Minesweeper games require an educated guess to be made. In the end, my solution was capable of solving 8x8 boards with 75% accuracy, and 16x16 with 60% accuracy.

ML Kaggle Competition

June 2020

This competition was a classwide competition for my Machine Learning & Data Mining Class. In this, my group and I were given a set of training data and had to come up with an ML model for the testing data. This project thoroughly summed up all our findings from the class and had a very straightforward usecase. My group's plan was to each come up with a model and after we have optimized each, and then "stack" the models. The models used were bagging (Random Forest) and boosting models (AdaBoost and Gradient Boost). The reason we chose both bagging and boosting was to decrease our final method's bias and variance since each model specializes in one. To optimize each model, we ran a 3 fold cross validation manipulating each parameter. We combined the models by utilizing a weighted mean which consisted of a bagging weight of 60% and a boosting weight of 40%. In the end, this led us to obtain a 79.903% accuracy for the testing data, one of the top five in the class. This project allowed me to soak in the brainstorming and implementation process that takes place to perfectly optimize a model and gave me a glimpse of how a scenario would be in the real world.
